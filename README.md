# Arabic → French Translator 🇸🇦 → 🇫🇷

##  Description

This project implements a deep learning model to translate Arabic sentences into French using a Seq2Seq (Encoder-Decoder) architecture with LSTM.

The model is trained on a small custom dataset and demonstrates the fundamentals of Neural Machine Translation (NMT).

---

##  Model Architecture

* Encoder-Decoder (Seq2Seq)
* LSTM layers
* Embedding layers
* Tokenization & Padding
